Help

Guideline



How to create new Election

This feature help Admin to create and conduct election.

  • Click on CREATE NEW option
  • Enter details about election.
  • Enter details about public(which take part in elections).
  • Enter details about candidates(which can be from public or outsider).
  • Write notification.
  • Send notification to voters.

How to cast Vote

This feature help Voter to cast vote successfully.

  • Click on CAST Vote option.
  • Login using your email(on which you received notification).
  • Use ID(giving in notification) to cast vote.
  • You can use your ID only one time.

How to check result of Election.

This feature help the user(admin/voter/candidate) to see election result.

  • Click on Result option.
  • Result will be displayed if election is successfully conducted.
  • Candidate name and position with total votes with be display.
  • You can also share result to other.

How to see previous Elections detail.

This feature help the user(admin/voter/candidate) to see previous election detail.

  • Go to Setting options.
  • Click on history.
  • You will be able to see all previous elections detail.

How to see status of notifications.

This feature help Admin to check notifications status.

  • Click on Status option.
  • Status of each notification will be display(send,not send).
  • Status of each voter response will be display(cast,not cast).
  • Status of election will be display(remaining time).

How to change Account details.

This feature help Admin to change or update account detail.

  • Go to Setting.
  • Click on Account option.
  • Make changes/update.
  • You can also Add another account.
  • If you no longer need this account you can also delete it.
